# TODO Задание 4. Приблизительное число π

Напишите програму, вычисляющую примерное значение числа π по выбранной пользвателем формуле:
1. Формула Виета для приближения числа π
2. Формула Валлиса
3. Ряд Лейбница

Формулы в общем виде можно найти в Интернете.

Программа должна предоставить пользователю выбор одной из формул и запросить точность, до какого знака после запятой
производить вычисления. Результат вывести на экран с заданной точностью.
